Welcome to **Escaping the Haunted House: A Zork Style Adventure**, a text-based adventure game built in C++.  
You are trapped in a haunted house. The only way out is by exploring rooms, collecting key items, and solving simple puzzles.

---

Extract the zip (and double-click `Zork.exe` to play.

## How to Play

- Use your keyboard to type commands and explore the world.
- Common inputs:
  - `NORTH`, `EAST`, `SOUTH`, `WEST` to move.
  - `Y` / `N` to make decisions (e.g., picking up items).
  - `QUIT` to exit the game.
